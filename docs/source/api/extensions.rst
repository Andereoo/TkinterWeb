TkinterWeb Extensions Documentation
===================================

The following objects are extensions to the :class:`~tkinterweb.TkinterWeb` widget and are largely internal. You will (and should) likely never need to access them, but they are described here just in case.

It is likely that over time more methods from the :class:`~tkinterweb.TkinterWeb` widget will be moved here.
